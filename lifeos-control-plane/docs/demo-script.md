# Demo Script (3-5 min)

1. Open dashboard and paste prompt:
   "I had a project sync today. Draft a polite follow-up email, suggest two meeting slots next week based on my preferences, create a task checklist from the meeting outcome, and summarize the top open GitHub issues in the project repo."
2. Submit and open workflow details.
3. Show planned steps with risk and policy decisions.
4. Show simulation previews and estimated latency/cost.
5. Open trace page to show lifecycle event timeline.
6. Open memory and policies pages to demonstrate governance.
7. Run one evaluation from evaluations page.
